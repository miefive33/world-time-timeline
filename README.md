# World Timeline Clock

各国の現在時刻を、ふつうの「時計の並び」ではなく **タイムライン形式** で表示するシンプルな Web アプリです。

GitHub Pages にそのまま置ける **HTML / CSS / JavaScript の静的構成** です。  
「ミラノの朝は日本の夕方」みたいな関係を、時計より直感的に読めるようにしています。

---

## 主な機能

- 日本 / シカゴ / ミラノ / ロンドン / ベトナムを初期表示
- 左側に都市リスト
- 各都市の現在時刻をデジタル表示
- 右側に 24 時間タイムライン表示
- 昼と夜を色分け
- タイムラインを左右ドラッグすると、すべての都市の時刻が連動
- `Now` ボタンで現在時刻に戻る
- 都市の追加 / 削除
- 都市の並び替え（ドラッグ & ドロップ）
- `localStorage` による状態保存

---

## 画面構成

- **Cities**: 国 / 都市の一覧
- **Digital**: 選択中の時刻基準でのデジタル時刻
- **Timeline**: 24時間の見える化エリア

## 使い方

### 1. ローカルで開く
`index.html` をブラウザで開くだけで動きます。

### 2. GitHub Pages で公開する
このフォルダを GitHub に push して、Pages を有効化してください。
